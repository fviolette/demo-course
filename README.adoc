
== Run development mode

[source,bash,attributes]
----
docker run -d --rm\
  --name=demo-course-dev \
  -p 3000:3000 \
  -v $PWD:/usr/src/app:Z \
  quay.io/redhat-scholars/course-site-builder dev
----

== Generate Publishing Site

[source,bash,attributes]
----
docker run -it \
  --name=demo-course-dev \
  -p 3000:3000 \
  -v $PWD:/usr/src/app \
  quay.io/redhat-scholars/course-site-builder site -p site.yml
----
